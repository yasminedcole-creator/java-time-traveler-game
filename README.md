# Java Time Traveler Game

A text-based adventure game built in Java that demonstrates
object-oriented programming, command parsing, and game state management.

## Project Structure
- App.java – Main entry point and game loop
- GameManager.java – Controls overall game logic and flow
- CommandSystem.java – Parses and executes player commands
- Entity.java – Base class for in-game entities
- Item.java – Represents collectible game items
- Location.java – Defines explorable locations

## Technologies
- Java
- Object-Oriented Programming (OOP)
- Command-line input

## Skills Demonstrated
- Multi-class architecture
- Separation of concerns
- Control flow and state management
